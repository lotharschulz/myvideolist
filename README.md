## Local Development
```shell script
./gradlew browserDevelopmentRun
```

```shell script
./gradlew browserDevelopmentRun --continuous
```

## Test
```shell script
./gradlew browserTest
```

```shell script
./gradlew check
```

## Pack
```shell script
./gradlew browserDevelopmentWebpack
```

```shell script
./gradlew browserProductionWebpack
```
find the files in `output` directory and upload it to your website.
